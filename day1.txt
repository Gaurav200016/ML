Lists
A list stores multiple items.


fruits = ["apple", "banana", "cherry"]
print(fruits[0])   # apple
fruits.append("orange")
print(fruits)

-------------------------------------------------------------------------
Dictionaries

Key-value pairs — great for structured data.


person = {"name": "Gauraw", "role": "ML Engineer", "experience": 0}
print(person["name"])
person["experience"] = 1
print(person)

If-Else

score = 85
if score >= 90:
    print("Excellent")
elif score >= 70:
    print("Good")
else:
    print("Needs Improvement")

Loops

for fruit in fruits:

count = 0
while count < 3:
    print("Looping...", count)
    count += 1
